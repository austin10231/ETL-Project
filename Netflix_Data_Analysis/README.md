# Netflix Data Cleaning and Analysis

This project demonstrates an **end-to-end data analysis workflow** using the Netflix Movies and TV Shows dataset.

## Overview
The dataset is cleaned, transformed, and analyzed using **SQL and Python**.  
The project includes data normalization, exploratory analysis, and SQL-based insights.

## Tech Stack
- SQL (PostgreSQL)
- Python
- Pandas

## Dataset
Netflix Movies and TV Shows dataset from Kaggle.

## Key Analysis
- Directors with the most movies
- Countries producing the most comedy movies
- Yearly content additions
- Genre distribution analysis
